# LLM Wiki Agents

LLM Wiki pattern: an agent maintains a structured Markdown knowledge base instead of relying only on chat history or opaque RAG indexes.

## Core Ideas

- Plain Markdown as durable memory.
- Raw/source/concept/synthesis separation.
- Ingest, query, lint, publish operations.
- Human-readable structure first; embeddings/search can be added later.
